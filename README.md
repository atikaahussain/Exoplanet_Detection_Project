Project Setup Instructions:

    Clone the repo

    Create a local environment: Run python3 -m venv venv in the project root.

    Activate the environment:

        Windows: .\venv\Scripts\activate

        Mac/Linux: source venv/bin/activate

    Install dependencies: Run pip install -r requirements.txt.

    Jupyter Kernel: If using Jupyter, run python -m ipykernel install --user --name=exoplanet_project
